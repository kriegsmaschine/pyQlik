Query Instructions:


Patient Parameters:

    To view results of single cohort:
    • select cohort
    • select gender
    • select Compare Field: NA
    To view cohort comparison results:
    • select multiple cohorts
    • select multiple genders
    • select desired parameter in Compare Field to compare
      • unselected Compare Field category is rolled into total counts

    To make multiple selections per field use CTRL + select or SHIFT + select

Clinical Data Parameters

  • select desired MSI values, leave blank to select all
  • select desired Vital status, leave blank to select all
  • select Days To Death, leave blank to select all
    • To choose values greater than or less than desired value
      choose a single value and select the appropriate radio button

    Values of -1 are place holders for Unknown values

Expression Data Parameters

  • Choose desired gene to analyze
  • Choose data representation
